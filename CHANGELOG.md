# Changelog

## 2.0.0 (2026-02-13)

Initial public release.

- 6 MCP tools: AoT, AoT-light, atomcommands, export_graph, generate_visualization, check_approval
- 5 atom types with confidence thresholds (Premise, Reasoning, Hypothesis, Verification, Conclusion)
- Interactive D3.js force-directed visualization with approve/reject workflow
- CLI flags: --mode, --max-depth, --no-viz, --no-approval, --output-dir, --downloads-dir
- Docker and Smithery support
- 121 tests
